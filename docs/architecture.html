Step & Reward
Shot Step
1. Decode force/spin, apply noise
2. Simulate shot (pool_sim)
3. Check 8-ball loss conditions
4. Detect fouls (scratch, wrong ball,
   no contact, no rail)
5. Remove pocketed, update positions
6. Assign groups if open table
7. Switch player or continue
   Foul → awaiting_placement
Placement Step
1. Denormalize (x, y) to table
2. Nudge away from ball overlaps
3. Clamp to valid region
4. Place cue ball
5. Compute placement reward
6. Return obs for shot selection
BIH turn = 2 steps total
Reward Signal
+0.15 per own ball pocketed
-0.15 per opponent ball pocketed
+0.10 cleared all own group
-0.40 any foul
+1.0 win (legal 8-ball pocket)
-1.0 loss (8-ball violation)
±0.05 shape bonus (ease of next shot)
±0.06 placement shaping (shot ease)
